UseUnicode

Get wether the default codepage is set to UTF-8. Setting this property to false will set the CodePage to SystemDefaultCodePage

Declaration

Remarks

/// Get OEM codepage from NetFX, which parses the NLP file with culture info table etc etc. But sometimes it yields the special value of 1 which is nicknamed CodePageNoOEM in System.Text.Encoding sources (might also mean CP_OEMCP, but Encoding puts it so). This was observed on Ukranian and Hindu systems. Given this value, System.Text.Encoding.GetEncoding(System.Int32) throws an System.ArgumentException. So replace it with ICSharpCode.SharpZipLib.Zip.ZipStrings.FallbackCodePage, (IBM 437 which is the default code page in a default Windows installation console.
